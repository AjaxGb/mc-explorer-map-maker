# Minecraft Explorer Map Maker

https://www.soltoder.com/mc-explorer-map-maker/

![Example screenshot of a generated explorer map being held in game](screenshot.png)

A tool for making blank explorer maps, similar to vanilla Minecraft. The logic for building the map image is
copied from the vanilla game, so this should be fully "accurate" for whatever body of water you draw or upload.

Downloaded map files should go in `<save>/data/map_?.dat` for versions before 26.1, and in `<save>/data/minecraft/maps/?.dat`
for 26.1 and later, where `?` is the map ID number. All versions from 1.11 forward should be supported;
please open an [issue](https://github.com/AjaxGb/mc-explorer-map-maker/issues/new) if you run into any problems.
